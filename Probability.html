The Second Floor: Probability

  • Definition
    • Sample Space: set of all possible outcomes
    • Event: subset of sample space
    • Probability: relative frequency over infinite trials
    • Exclusive Events
  • Notation
    • and, or, complement
  • Probabilistic Law
    • Multiplication
      • Independent Events: P(A ∧ B) = P(A) × P(B)
    • Addition
      • Pr(A ∨ B) = Pr(A) + Pr(B) - Pr(A ∧ B)
      • Independent Events: Pr(A ∨ B) = Pr(A) + Pr(B) × (1-Pr(A))
    • Conditional Probability
      • Pr(A|B) = Pr(A ∧ B) / Pr(B)
      • Independent Events: Pr(B|A) = Pr(B) = Pr(B|¬A)
      • Relative Risk of B given A: Pr(B|A) / Pr(B|¬A)
    • Total-Probability Rule
      • Pr(B) = Pr(B|A)Pr(A) + Pr(B|¬A)Pr(¬A)
      • Pr(B) = Σ[Pr(B|A=aᵢ) × Pr(A=aᵢ)] (i=1,2,3...n)
    • Generalized Multiplication Law
      • Pr(A₁∧A₂∧...∧Aₙ) = P(A₁) × P(A₂|A₁) × P(A₃|A₁∧A₂) × ... × P(Aₙ|A₁∧A₂∧...∧Aₙ₋₁)
  • Bayes' Rule and Screening Tests
    • Predictive value positive (PV+): Pr(disease|test+)
    • Predictive value negative (PV-): Pr(¬disease|test-)
    • Sensitivity
      • true⁺ / (true⁺ + false⁻)
      • Pr(test+|disease)
    • Specificity
      • true⁻ / (true⁻ + false⁺)
      • Pr(test-|¬disease)
    • Bayes' Rule
      • A: test⁺, B: disease⁺
      • PV+ = Pr(B|A) = [Pr(A|B) × Pr(B)] / [Pr(A|B)Pr(B) + Pr(A|¬B)Pr(¬B)]
      • PV+ = [sensitivity × Pr(B)] / [sensitivity×Pr(B) + (1-specificity)×Pr(¬B)]
      • PV- = [specificity × Pr(¬B)] / [specificity×Pr(¬B) + (1-sensitivity)×Pr(B)]
    • Generalized Bayes' Rule
      • Pr(Bᵢ|A) = [Pr(A|Bᵢ)Pr(Bᵢ)] / Σ[Pr(A|Bⱼ)Pr(Bⱼ)] (j=1,2,3...k)
  • Bayesian Inference
    • Prior probability
    • Posterior probability: include PV+, PV-...
  • ROC Curve: Receiver Operating Characteristic
    • x axis: 1-specificity; false⁺ rate
    • y axis: sensitivity; true⁺ rate
    • AUC: Area Under Curve
      • AUC > 0.9: good classifier
  • Prevalence and Incidence
    • Incidence: cumulative incidence
