The Second Floor: Overview of Supervised Learning

  • Use the inputs (predictors) to predict the output (response)
  • Variable Types and Terminology
    • Quantitative output: regression
    • Qualitative output: classification
    • Dummy Variable
    • X predict 'ŷ', categorical output transformed into dummy variable
  • Two Prediction Approaches
    • Least Squares: linear model
      • ŷ = b₀ + b₁X₁ + b₂X₂ + ... + bₚXₙ
      • ŷ = XTβ
      • b₀: intercept, bias in machine learning
      • Residual Sum of Squares (RSS):
        • RSS = Σ(yᵢ - ŷ)²
        • RSS = (y - Xβ)T(y - Xβ)
        • Differentiating w.r.t. β gives the normal equation
        • XT(y - Xβ) = 0
        • β = (XTX)-1 XTy
    • Nearest Neighbors: k-nearest-neighbor prediction rule
      • ŷ = 1/k Σyᵢ (xᵢ ∈ Nₖ(x))
      • Find k-nearest neighbors of x in training set and then vote
  • From Least Squares to Nearest Neighbors
