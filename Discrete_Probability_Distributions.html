The Third Floor: Discrete Probability Distributions

  • Random Variable: a function that assigns numeric values to different events in a sample space
    • Discrete
    • Continuous
  • Probability-Mass Function (PMF) for Discrete Random Variables
    • PMF: Pr(X=x); theoretical
  • Expected Value
  • Variance: Var(X) = E[(X-E(X))2]
  • Standard Deviation
  • Cumulative-Distribution Function
  • Permutation and Combination
    • Permutation: nPk = n! / (n-k)!
    • n!: n factorial
    • Combination: nCk = n! / (k! × (n-k)!) = nPk / k!
  • Binomial Distribution
    • n trials, two possible outcomes (success or failure)
    • Pr(X=k) = C(n,k) × pk × (1-p)n-k = Pr(Y=n-k)
    • Expected Value: E(X) = n × p
    • Variance: Var(X) = n × p × (1-p)
  • Poisson Distribution
    • k events occurring in time period t; λ = expected events per unit time
    • Pr(X=k) = e-μ × μk / k!, k=0,1,2,3...
    • μ = λ × t
    • Example
      • μ = 4.6, t = 1 year
      • λ = μ × t = 4.6 deaths/year
      • If Δt = 6-month: μ = 2.3
      • Pr(X=k) = e-2.3 × 2.3k / k!
    • Expected Value: E(X) = μ
    • Variance: Var(X) = μ
  • Poisson Approximation to the Binomial Distribution
    • Binomial Distribution with large n and small p
    • Poisson Distribution with μ = np (easier to work with)
